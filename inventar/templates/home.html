Verwendung:

Fall 1: Suchen eines Gegenstands

  1. Den gesuchten Gegenstand links in die Suchmaske eingeben.
  2. Die Suchergebnisse erscheinen im Hauptfenster.
  3. Suchergebnis auswählen.
  4. Der Pfad zum Gegenstand ist als Baum dargestellt.

Fall 2: Zurückbringen eines Gegenstands

  1. Alle Gegenständen ist mindestens ein viertstelliger Code zugewiesen.
  2. Dieser kann einfach links in das Code Suchfeld eingetragen werden.
  3. Man gelangt so direkt zu der Info Seite des Gegenstands.
  4. Unter dem Pfad lässt sich der Ablageort des Gegenstands finden.

Fall 3: Einräumen einer Kiste/eines Schrankes/...

  1. Die Seite des Gegenstands öffnen, der befüllt werden soll.
  2. Dort gibt es ein Feld "Kinderelemente", welche alle Gegenstände listet, die Unterlemente sind.

Sonstige Erklärungen

  1. Einige Gegenstände haben temporär einen gesonderten Ort. Der Pfad zu diesem wird direkt unter dem normalen Pfad dargestellt.
  2. Ob Gegenstände frei zur Verfügung stehen lässt sich an dem "in Benutzung" Feld erkennen (Das Icon sind zwei Zahnräder).
  3. Bei einigen Gegenständen handelt es sich um Dauerleihgaben von Mitgliedern. Dies lässt sich am Besitzer Feld erkennen. Das Feld ist nur bei Gegenständen vorhanden, welche auch einen Besitzer haben (Das Icon ist eine Person).
